# Titanic Data Analysis: Exploratory Data Analysis & Visualisation

This repository contains a Jupyter Notebook implementing a complete Exploratory Data Analysis (EDA) workflow on the famous Titanic Survival Dataset using Python.

## 📈 Key Insights Covered:
- **Data Cleaning:** Safely handled missing values for Age and Embarked columns and removed duplicates.
- **Feature Engineering:** Created new meaningful variables like `Age Group`, `FamilySize`, and `IsAlone`.
- **Advanced Visualisations:** Included Bar charts, Age distribution Histograms, Fare Box plots, Correlation Heatmaps, Violin plots, and FacetGrids using Matplotlib and Seaborn.

## 🛠️ Requirements:
Run the notebook using Jupyter Notebook or VS Code with the following libraries installed:
`pip install numpy pandas matplotlib seaborn`
